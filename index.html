Powerful Features to Drive Better Search

TestMySearch provides a complete toolkit for search analysis and optimization.

A/B Testing Platform

Feed the system thousands of queries and multiple configurations. By morning, get a full report with charts, metrics, and a clear conclusion on which performs better.

Comprehensive Metrics

Calculate NDCG@k, MAP, Precision, Recall, and dozens more. Interactive visualizations make it easy to understand the "why" behind the numbers.

Virtual Search Assessor

Leverage LLMs to evaluate how well each document matches a query. Our non-trivial process of chunking, embeddings, and evaluation delivers reliable relevance scores.

Query Analysis & Clustering

Analyze and group similar queries to uncover user intent patterns. For example, find brand name variations that should be treated as equivalents.

A Clear Path to Better Relevancy

Our streamlined process makes search testing systematic and repeatable.

1

Upload & Configure

Provide your query sets, ground truth data (evalsets), and connect your search engine configurations (Solr, Elasticsearch, etc.).
2

Run & Analyze

Execute batch runs. TestMySearch fetches results, calculates IR metrics against your ground truth, and optionally gets LLM judgements on performance.
3

Visualize & Decide

Review detailed per-query and summary reports. Compare configurations side-by-side and make informed decisions backed by data.
